# corpus — ベンチマーク用スキルコーパス

実測は「実物スキル10本＋合成スキル10本」の計20本で行いました。このリポジトリに入っているのは**合成10本（`synth/`）のみ**です。実物10本は作成者の本番運用スキル（私物・一部は他者著作物のコピー）のため公開しません。したがって、同梱物だけでは論文値の12条件マトリクスは完全再現できません——**自分のスキルを10本足せば同一条件になります**。

## 合成10本の設計

各スキルは「説明文の質」「相互類似度」「本文長」を意図的に振ってあります。例：`keihi-seisan`（経費チェック）と`shiwake-henkan`（仕訳変換）は境界を明記した対、`meibo-kanri`は説明文をわざと曖昧に、`data-bunseki`はわざと過広（「なんでも分析」）にして誤起動を誘発する設計です。

## 索引の作り方

コーパスを足したり替えたりしたら、索引を再生成します：

```bash
python bench/build_index.py
```

`corpus/{real,synth}/*/SKILL.md` を走査して `corpus/index.json` を作ります（ルーターとベンチの両方がこれを読みます）。
